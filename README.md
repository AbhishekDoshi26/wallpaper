# wallpaper 🖼

Make the most of your display with beautiful wallpapers. ✨

## Getting Started

- Install the apk file 📲

- Open the app 📱

- Set your favourite wallpaper to home/lock/both screen 📳

- That's it! Your home screen just got a cool new refreshing makeover 😏
